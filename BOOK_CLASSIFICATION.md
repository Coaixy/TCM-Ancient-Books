## 分析口径

- 仓库根目录共有 701 个可用 `.txt` 文本文件；未把 `.baiduyun.downloading` 和 `.cfg` 临时文件纳入统计。

- 分类主要依据书名关键词，辅以抽样内容确认文本格式和主题；一书多属时按主要主题归入一个类别。
- README 已提示部分 OCR/网上文本可能缺段或不完整，因此本表适合作为浏览索引，不宜直接视为版本完整性结论。

- 少数文件行数很短，可能是节选、残本或 OCR/下载不完整文本；后续若做知识库，应先做完整性和重复性校验。
