# TCM-Ancient-Books

中医药古籍 OCR 文本数据库。

## 项目结构

```text
.
├── BOOK_CLASSIFICATION.md        # 全量书目初步分类
├── BOOK_INDEX_AZ.md              # 全量书目 A-Z 索引
├── TOPIC_SUBDIVISION.md          # 专题细分索引
├── 原文/
│   ├── H/
│   ├── A/
│   ├── B/
│   ├── ...
│   ├── Z/
│   └── #/
└── 专题细分/
    ├── README.md                 # 专题目录说明
    ├── 中医体质分类与判定.md       # 体质分类与判定数据
    ├── 01-中医食疗/
    ├── 02-穴位应用/
    ├── 03-中医运动/
    ├── 04-作息与情绪管理/
    ├── 05-慢病管理/
    ├── 06-体重管理/
    └── 07-中医体质调养/
```


## 数据说明

- `原文/` 中保留全量 OCR 文本，目前约 707 本，按书名拼音首字母分目录存放。
- `专题细分/` 中按主题复制了精选文本，每个专题 10 本核心资料。
- `BOOK_CLASSIFICATION.md` 用于查看全量书目的粗分类。
- `BOOK_INDEX_AZ.md` 用于按拼音首字母检索书目，并链接到 `原文/` 下对应文本。
- `TOPIC_SUBDIVISION.md` 用于查看专题候选和精选清单。

## 注意

部分 OCR 文本可能存在错字、缺段、残本或版本不明等问题，严肃引用前需要人工校对。
