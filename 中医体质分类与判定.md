# 中医体质分类与判定（GB/T 46939-2025）

资料来源：

- 正式标准：GB/T 46939-2025《中医体质分类与判定》
- 发布日期：2025-12-31
- 实施日期：2026-04-01
- 主管部门、归口部门：国家中医药局

说明：

- 本文件用于健康管理系统建库和规则设计。
- 九种体质的详细特征以 GB/T 46939-2025 第 4 章为依据。
- 正式体质判定以附录 A 量表计分为主，舌象、面色、唇色、肤色等只作为辅助证据。
- A-I 型编码沿用旧资料，便于系统兼容；正式标准主要使用“平和质、气虚质”等体质名称。

## 一、体质类型总览

| 编码 | 体质类型 | 类型属性 | 核心特征 |
|---|---|---|---|
| A | 平和质 | 正常体质 | 阴阳气血调和，体态适中，面色红润，精力充沛 |
| B | 气虚质 | 偏颇体质 | 元气不足，疲乏、气短、自汗 |
| C | 阳虚质 | 偏颇体质 | 阳气不足，畏寒怕冷，手足不温 |
| D | 阴虚质 | 偏颇体质 | 阴液亏少，口燥咽干，手足心热 |
| E | 痰湿质 | 偏颇体质 | 痰湿凝聚，形体肥胖，腹部肥满，口黏苔腻 |
| F | 湿热质 | 偏颇体质 | 湿热内蕴，面垢油光，口苦，苔黄腻 |
| G | 血瘀质 | 偏颇体质 | 血行不畅，肤色晦暗，舌质紫暗 |
| H | 气郁质 | 偏颇体质 | 气机郁滞，神情抑郁，忧虑脆弱 |
| I | 特禀质 | 偏颇体质 | 禀赋不耐，以过敏反应等为主要特征 |

## 二、九种体质详细特征

### 平和质（A 型）

主要特征：阴阳气血调和，以体态适中、面色红润、精力充沛等为主要特征。

形体特征：体形匀称健壮。

常见表现：面色、肤色润泽，头发稠密有光泽，目光有神，鼻色明润，嗅觉通利，唇色红润，不易疲劳，精力充沛，耐受寒热，胃纳佳，睡眠良好，二便正常，舌淡红，苔薄白，脉和缓有力。

心理特征：性格随和开朗。

发病倾向：平素患病较少。

对外界环境适应能力：对自然环境和社会环境适应能力较强。

舌面辅助判断：

- 舌象：舌淡红，苔薄白。
- 面象：面色和肤色润泽，目光有神，鼻色明润，唇色红润。
- 辅助要点：舌面整体清爽、颜色适中，且无明显寒、热、痰湿、瘀暗表现。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| A1 | 精力是否充沛 | 正向 |
| A2 | 是否容易疲乏 | 逆向 |
| A3 | 是否闷闷不乐、情绪低沉 | 逆向 |
| A4 | 是否比一般人耐受不了寒冷、冷空调或电扇 | 逆向 |

### 气虚质（B 型）

主要特征：元气不足，以疲乏、气短、自汗等为主要特征。

形体特征：肌肉松软不实。

常见表现：平素语音低弱，气短懒言，容易疲乏，精神不振，易出汗，舌淡红，舌边有齿痕，脉弱。

心理特征：性格内向，不喜冒险。

发病倾向：易患感冒、内脏下垂、肥胖、骨质疏松、老年便秘、慢性疲劳综合征、焦虑症等病；病后康复缓慢。

对外界环境适应能力：不耐受风、寒、暑、湿、热邪。

舌面辅助判断：

- 舌象：舌淡红，舌边可见齿痕。
- 面象：面部特异性不强，可结合神疲、少气懒言、自汗等表现判断。
- 辅助要点：齿痕舌加疲乏、气短、自汗，更支持气虚质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| B1 | 是否容易疲乏 | 正向 |
| B2 | 是否容易气短、呼吸短促、接不上气 | 正向 |
| B3 | 是否容易心慌 | 正向 |

### 阳虚质（C 型）

主要特征：阳气不足，以畏寒怕冷、手足不温等虚寒表现为主要特征。

形体特征：肌肉松软不实。

常见表现：平素畏冷，手足不温，喜热饮食，精神不振，舌淡胖嫩，或舌边有齿痕，脉沉迟。

心理特征：性格多沉静、内向。

发病倾向：易患痰饮、肿胀、泄泻、骨质疏松、慢性疲劳综合征、原发性痛经、膝关节炎等病；感邪易从寒化。

对外界环境适应能力：耐夏不耐冬；易感风、寒、湿邪。

舌面辅助判断：

- 舌象：舌淡胖嫩，或舌边有齿痕。
- 面象：面部特异性不强，可结合畏寒、手足不温、喜热饮食判断。
- 辅助要点：舌体胖嫩偏淡、齿痕，加怕冷和手足不温，更支持阳虚质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| C1 | 胃脘部、背部或腰膝部是否怕冷 | 正向 |
| C2 | 是否感到怕冷、衣服比别人穿得多 | 正向 |
| C3 | 是否比一般人耐受不了寒冷、冷空调或电扇 | 正向 |

### 阴虚质（D 型）

主要特征：阴液亏少，以口燥咽干、手足心热等虚热表现为主要特征。

形体特征：体形偏瘦。

常见表现：手足心热，眼睛干涩，口燥咽干，鼻微干，喜冷饮，大便干燥，舌红少津，脉细数。

心理特征：性格活泼，性情急躁，外向好动。

发病倾向：易患虚劳、遗精、不寐、骨质疏松症、糖尿病、高血压、偏头痛、功能性便秘、肌少症、干眼症等病；感邪易从热化。

对外界环境适应能力：耐冬不耐夏；不耐受暑、热、燥邪。

舌面辅助判断：

- 舌象：舌红少津。
- 面象：鼻微干，口唇干，面部两颧潮红或偏红。
- 辅助要点：红、干、少津与虚热表现一致，结合身体或脸上发热感更支持阴虚质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| D1 | 是否感觉身体、脸上发热 | 正向 |
| D2 | 皮肤或口唇是否干 | 正向 |
| D3 | 面部两颧是否潮红或偏红 | 正向 |

### 痰湿质（E 型）

主要特征：痰湿凝聚，以形体肥胖、腹部肥满、口黏苔腻等为主要特征。

形体特征：体形肥胖，腹部肥满松软。

常见表现：面部皮肤油脂较多，多汗且黏，胸闷，痰多，口黏腻或甜，喜食肥甘甜黏，舌胖大，苔腻，脉滑。

心理特征：性格偏温和、稳重，善于忍耐。

发病倾向：易患消渴、中风、胸痹、失眠、代谢综合征、肥胖、高血压、高脂血症、脂肪肝、肝硬化、高尿酸血症、肠息肉、多囊卵巢综合征等病。

对外界环境适应能力：对梅雨季节及潮湿环境适应能力差。

舌面辅助判断：

- 舌象：舌胖大，苔腻。
- 面象：面部皮肤油脂较多，可伴体形肥胖、腹部肥满松软。
- 辅助要点：胖大舌、腻苔、面油、腹满共同出现时，更支持痰湿质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| E1 | 是否感到身体沉重、不轻松或不爽快 | 正向 |
| E2 | 腹部是否肥满松软 | 正向 |
| E3 | 嘴里是否有黏黏的感觉 | 正向 |

### 湿热质（F 型）

主要特征：湿热内蕴，以面垢油光、口苦、苔黄腻等为主要特征。

形体特征：形体中等或偏瘦。

常见表现：面垢油光，易生痤疮，口苦口干，身重困倦，大便黏滞不畅或燥结，小便短黄，男性易阴囊潮湿，女性易带下色黄量多，舌质偏红，苔黄腻，脉滑数。

心理特征：容易心烦急躁。

发病倾向：易患痤疮、黄疸、热淋、不寐、消渴、高尿酸血症、非酒精性脂肪性肝病、肝硬化、肠息肉等病。

对外界环境适应能力：对夏末秋初湿热气候，潮湿或气温偏高环境较难适应。

舌面辅助判断：

- 舌象：舌质偏红，苔黄腻。
- 面象：面垢油光，易生痤疮。
- 辅助要点：红舌、黄腻苔、面油、痤疮共同出现时，更支持湿热质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| F1 | 面部或鼻部是否有油腻感、油亮发光 | 正向 |
| F2 | 小便时尿道是否有发热感、尿色是否浓或深 | 正向 |
| F3F | 女性：带下是否色黄 | 正向 |
| F4M | 男性：阴囊部位是否潮湿 | 正向 |

计分说明：湿热质正式表中第 3 题限女性回答，第 4 题限男性回答；实际计分时按性别二选一，因此湿热质实际计分条目数为 3。

### 血瘀质（G 型）

主要特征：血行不畅，以肤色晦暗、舌质紫暗等为主要特征。

形体特征：胖瘦均见。

常见表现：肤色晦暗，色素沉着，容易出现瘀斑，口唇色暗，舌暗或有瘀点，舌下络脉紫暗或增粗，脉涩。

心理特征：易烦，健忘。

发病倾向：易患癥瘕、痛证、血证、消渴、心血管疾病（高血压、动脉粥样硬化、冠心病）、突发性耳聋、子宫内膜异位症等。

对外界环境适应能力：不耐受寒邪。

舌面辅助判断：

- 舌象：舌暗或有瘀点，舌下络脉紫暗或增粗。
- 面象：肤色晦暗，色素沉着，容易出现瘀斑，口唇色暗。
- 辅助要点：紫暗、瘀点、唇暗、褐斑是血瘀质的重要外观证据。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| G1 | 身体上是否有疼痛处 | 正向 |
| G2 | 面色是否晦暗，或是否容易出现褐斑 | 正向 |
| G3 | 口唇颜色是否偏暗 | 正向 |

### 气郁质（H 型）

主要特征：气机郁滞，以神情抑郁、忧虑脆弱等为主要特征。

形体特征：形体瘦者为多。

常见表现：神情抑郁，情感脆弱，烦闷不乐，舌淡红，苔薄白，脉弦。

心理特征：性格内向不稳定、敏感多虑。

发病倾向：易患脏躁、梅核气、百合病、郁证（抑郁）、瘿瘤、不寐、认知衰弱等。

对外界环境适应能力：对精神刺激适应能力较差；不适应阴雨天气。

舌面辅助判断：

- 舌象：舌淡红，苔薄白。
- 面象：面部特异性不强，可见神情抑郁、情绪低落。
- 辅助要点：气郁质舌面线索相对弱，主要依赖情志条目和访谈。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| H1 | 是否闷闷不乐、情绪低沉 | 正向 |
| H2 | 是否容易精神紧张、焦虑不安 | 正向 |
| H3 | 是否多愁善感、感情脆弱 | 正向 |

### 特禀质（I 型）

主要特征：禀赋不耐，以过敏反应等为主要特征。

形体特征：一般无特殊。

常见表现：常见哮喘、风团、咽痒、鼻塞、喷嚏等，多见齿痕舌。

心理特征：容易伴随焦虑、紧张。

发病倾向：易患哮喘、荨麻疹、花粉症及药物过敏等。

对外界环境适应能力：适应能力差，对易致敏季节适应能力差，易引发宿疾。

舌面辅助判断：

- 舌象：多见齿痕舌。
- 面象：一般无固定面色特征，可结合鼻塞、喷嚏、风团、抓痕等过敏表现。
- 辅助要点：齿痕舌加喷嚏、鼻塞、风团、抓痕等过敏线索，更支持特禀质。

量表条目：

| 条目 ID | 题目要点 | 计分 |
|---|---|---|
| I1 | 没有感冒时是否也会打喷嚏 | 正向 |
| I2 | 是否容易对药物、食物、气味、花粉、季节或气候变化过敏 | 正向 |
| I3 | 皮肤是否容易起荨麻疹、风团、风疹块、风疙瘩 | 正向 |
| I4 | 皮肤一抓是否就红，并出现抓痕 | 正向 |

### 3. 选项计分

所有条目均按近一年的体验和感觉回答。

| 分值 | 选项含义 |
|---:|---|
| 1 | 没有 / 根本不 |
| 2 | 很少 / 有一点 |
| 3 | 有时 / 有些 |
| 4 | 经常 / 相当 |
| 5 | 总是 / 非常 |

### 4. 逆向计分

只有平和质中标有逆向的 A2、A3、A4 需要先逆向计分，再计算平和质转化分。同一答案映射到 B1、H1、C3 时不逆向。

| 原答值 | 逆向分 |
|---:|---:|
| 1 | 5 |
| 2 | 4 |
| 3 | 3 |
| 4 | 2 |
| 5 | 1 |

### 5. 原始分和转化分

每种体质分别计算。

```text
原始分 = 各个条目分值相加
转化分 = [(原始分 - 条目数) / (条目数 * 4)] * 100
```

| 体质 | 条目数 |
|---|---:|
| 平和质 | 4 |
| 气虚质 | 3 |
| 阳虚质 | 3 |
| 阴虚质 | 3 |
| 痰湿质 | 3 |
| 湿热质 | 3 |
| 血瘀质 | 3 |
| 气郁质 | 3 |
| 特禀质 | 4 |

### 6. 国标判定标准

平和质为正常体质，其他 8 种体质为偏颇体质。

| 体质类型 | 条件 | 判定结果 |
|---|---|---|
| 平和质 | 平和质转化分 >= 60，且其他 8 种体质转化分均 < 30 | 是 |
| 平和质 | 平和质转化分 >= 60，且其他 8 种体质转化分均 < 40 | 基本是 |
| 平和质 | 不满足上述条件 | 否 |
| 偏颇体质 | 本体质转化分 >= 40 | 是 |
| 偏颇体质 | 本体质转化分 >= 30 且 < 40 | 倾向是 |
| 偏颇体质 | 本体质转化分 < 30 | 否 |

### 7. 系统输出建议

国标判定是分别判断 9 种体质，因此一个人可能有多个偏颇体质同时达到“是”。系统展示时建议分层表达，避免把所有达标体质都并列成主结果。

1. 先计算 9 种体质的转化分。
2. 按国标规则分别给出每种体质的“是”“基本是”“倾向是”“否”。
3. 如果有偏颇体质达到“是”，取转化分最高者作为主导体质；其他达到“是”的体质作为兼夹体质或复核线索。
4. 如果没有偏颇体质达到“是”，且平和质为“是”或“基本是”，主导体质输出平和质或基本平和质。
5. 如果没有偏颇体质达到“是”，但存在“倾向是”，取转化分最高者作为倾向体质，并建议结合舌象、面色或人工复核。
6. 如果 4 种及以上偏颇体质同时达到“是”或“倾向是”，建议提示复核答题，不能简单理解为同时拥有很多主导体质。

## 四、舌象与面相辅助判断

舌面判断不能替代量表评分，建议作为解释证据和复核提示。

| 体质 | 舌象线索 | 面部、唇色、肤色线索 | 辅助判断要点 |
|---|---|---|---|
| 平和质 | 舌淡红，苔薄白 | 面色和肤色润泽，目光有神，鼻色明润，唇色红润 | 舌面清爽、颜色适中，且偏颇表现少 |
| 气虚质 | 舌淡红，舌边可见齿痕 | 面部特异性不强 | 齿痕舌加疲乏、气短、自汗，更支持气虚 |
| 阳虚质 | 舌淡胖嫩，或舌边有齿痕 | 面部特异性不强 | 舌体胖嫩偏淡、齿痕加怕冷，更支持阳虚 |
| 阴虚质 | 舌红少津 | 鼻微干，口唇干，两颧潮红或偏红 | 红、干、少津加发热感，更支持阴虚 |
| 痰湿质 | 舌胖大，苔腻 | 面部皮肤油脂较多，常伴腹部肥满 | 胖大舌、腻苔、面油、腹满，更支持痰湿 |
| 湿热质 | 舌质偏红，苔黄腻 | 面垢油光，易生痤疮 | 红舌、黄腻苔、面油、痤疮，更支持湿热 |
| 血瘀质 | 舌暗或有瘀点，舌下络脉紫暗或增粗 | 肤色晦暗、色素沉着、瘀斑、唇色暗 | 紫暗、瘀点、唇暗、褐斑，更支持血瘀 |
| 气郁质 | 舌淡红，苔薄白 | 可见神情抑郁、情绪低落 | 舌面线索弱，主要依赖情志条目 |
| 特禀质 | 多见齿痕舌 | 可见鼻塞、喷嚏、风团、抓痕等过敏表现 | 齿痕舌加过敏表现，更支持特禀 |

图像识别或人工望诊建议输出三层结果：

| 层级 | 依据 | 输出 |
|---|---|---|
| 主判定 | 附录 A 量表转化分 | 平和质、偏颇体质、倾向体质、兼夹体质 |
| 辅助证据 | 舌象与面部标签 | 舌象支持阴虚质、面部油光支持湿热或痰湿倾向等 |
| 复核提示 | 问卷与图像冲突 | 问卷提示某体质但舌面证据不足，建议人工复核 |
